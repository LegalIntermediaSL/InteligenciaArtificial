# Bloque 27 — Evaluación y benchmarks de LLMs

Aprende a interpretar benchmarks públicos (MMLU, HumanEval, LMSYS Arena), crear evaluaciones propias para tu dominio específico, comparar modelos de forma rigurosa y tomar decisiones informadas sobre qué modelo usar en producción.
