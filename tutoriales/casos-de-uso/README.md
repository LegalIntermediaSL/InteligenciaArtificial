# Casos de uso prácticos

Proyectos completos que aplican los conceptos de los bloques anteriores a problemas reales. Cada caso de uso incluye el código completo, explicaciones paso a paso y sugerencias de extensión.

## Contenido

| # | Caso de uso | Descripción |
|---|---|---|
| 01 | [Chatbot con Claude API](./01-chatbot-claude-api.md) | Asistente conversacional con historial y system prompt |
| 02 | [Clasificación de texto](./02-clasificacion-texto.md) | Clasificar textos con LLMs y con ML clásico |
| 03 | [Resumen automático de documentos](./03-resumen-documentos.md) | Resumir documentos largos con estrategia de chunks |
| 04 | [Extracción de información de PDFs](./04-extraccion-pdfs.md) | Leer PDFs y extraer datos estructurados con IA |

## Requisitos

```bash
pip install anthropic openai pypdf python-dotenv tqdm scikit-learn
```

Cada caso especifica sus propias dependencias adicionales.

## Orden recomendado

01 → 02 → 03 → 04. Puedes leer cualquiera de forma independiente.
