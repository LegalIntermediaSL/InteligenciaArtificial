# Fundamentos de Inteligencia Artificial

Introducción a los conceptos base de la IA. Este bloque es el punto de partida recomendado antes de abordar cualquier otro bloque del proyecto.

## Contenido

| # | Tutorial | Descripción |
|---|---|---|
| 01 | [¿Qué es la Inteligencia Artificial?](./01-que-es-la-ia.md) | Definición, origen del término y panorama general |

- Tutoriales 01-03: no se requieren conocimientos previos. Contenido teórico y conceptual.
- Tutoriales 04-05: nivel intermedio. Se introduce notación matemática básica, pero siempre acompañada de intuición y ejemplos. No se requiere programación.

## Orden recomendado

Leer en orden numérico: 01 → 02 → 03 → 04 → 05.
