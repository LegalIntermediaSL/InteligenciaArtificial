# APIs de IA

Guías prácticas para integrar los principales proveedores de modelos de lenguaje en tus proyectos.

## Contenido

| # | Tutorial | Descripción |
|---|---|---|
| 01 | [API de Anthropic (Claude)](./01-api-anthropic-claude.md) | Primeros pasos, mensajes, herramientas y buenas prácticas |
| 02 | [API de OpenAI](./02-api-openai.md) | Primeros pasos, chat completions, funciones y embeddings |
| 03 | [Comparativa de proveedores](./03-comparativa-proveedores.md) | Precios, capacidades, contexto y cuándo elegir cada uno |
| 04 | [Google Gemini](./04-google-gemini.md) | Familia Gemini, contexto 1M tokens, multimodalidad, Vertex AI |
| 05 | [Mistral y Cohere](./05-mistral-cohere.md) | Mistral (open weights, Europa), Cohere (embeddings, reranking, RAG) |

## Requisitos

- Python 3.10+
- Cuenta y API key en el proveedor correspondiente
- `pip install anthropic openai google-generativeai mistralai cohere`

## Orden recomendado

01 → 02 → 03 → 04 → 05. Si solo trabajarás con un proveedor, lee solo su tutorial y el 03.
