# APIs de IA

Guías prácticas para integrar los principales proveedores de modelos de lenguaje en tus proyectos.

## Contenido

| # | Tutorial | Descripción |
|---|---|---|
| 01 | [API de Anthropic (Claude)](./01-api-anthropic-claude.md) | Primeros pasos, mensajes, herramientas y buenas prácticas |

| 03 | [Comparativa de proveedores](./03-comparativa-proveedores.md) | Precios, capacidades, contexto y cuándo elegir cada uno |

## Requisitos

- Python 3.10+
- Cuenta y API key en el proveedor correspondiente
- `pip install anthropic openai`

## Orden recomendado

01 → 02 → 03. Si solo trabajarás con un proveedor, lee solo su tutorial y el 03.
