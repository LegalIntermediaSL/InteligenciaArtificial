# Casos de uso avanzados

Aplicaciones de IA que combinan múltiples técnicas: structured output, embeddings, agentes con herramientas y análisis de datos. Cada tutorial incluye código completo listo para producción.

## Contenido

| # | Tutorial | Descripción | Nivel |
|---|---|---|---|
| 01 | [Generación y revisión de código](./01-generacion-revision-codigo.md) | Revisar código, generar tests y documentar con IA | Avanzado |
| 02 | [Búsqueda semántica con embeddings](./02-busqueda-semantica.md) | RAG sin dependencias pesadas — solo numpy y JSON | Avanzado |
| 03 | [Structured output con Instructor](./03-structured-output-instructor.md) | Extracción robusta de datos con Pydantic e Instructor | Avanzado |
| 04 | [Análisis de datos con IA](./04-analisis-datos-ia.md) | Agente analista con herramientas Python ejecutables | Avanzado |

## Requisitos

```bash
pip install anthropic openai instructor pydantic pandas matplotlib
```

Cada tutorial especifica sus propias dependencias adicionales.

## Orden recomendado

01 → 02 → 03 → 04. Puedes leer cualquiera de forma independiente.
