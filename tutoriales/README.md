# Tutoriales

Esta carpeta contiene los tutoriales del proyecto, organizados por temática.

## Estructura prevista

```
tutoriales/
├── fundamentos/          # Conceptos base de IA
├── llms/                 # Modelos de lenguaje (LLMs)
├── apis/                 # Uso de APIs de IA (Claude, OpenAI, etc.)
├── python-para-ia/       # Python aplicado a IA
└── casos-de-uso/         # Proyectos y ejemplos prácticos
```

Cada subcarpeta incluirá su propio `README.md` con descripción, requisitos y orden recomendado de lectura.
